## リード文

AI面接練習・人材マッチングプラットフォーム「導（みちびき）」（URL: https://michibiki.tech ）は、33分野に対応したAI模擬面接サービスを正式リリースいたしました。AIが面接官を務め、候補者一人ひとりの回答に応じて質問を動的に変化させる「適応型AI面接」を、完全無料・回数無制限で提供します。

---

## 背景と課題

転職・就職活動において、面接対策は合否を分ける最重要ステップです。しかし、効果的な面接練習を行うには「適切な面接官役を見つける」「本番と同等の緊張感で練習する」「客観的なフィードバックを得る」という3つのハードルがあり、多くの求職者が十分な準備ができないまま本番に臨んでいるのが現状です。

特に技術面接（コーディング面接、システム設計面接）においては、専門知識を持つ面接官役の確保が困難であり、有料のコーチングサービスも1回あたり数万円と高額です。

「導（みちびき）」は、この構造的な課題をAI技術で解決します。

---

## サービスの特徴

### 1. 適応型AI面接

従来のAI面接サービスが「あらかじめ用意した質問を順番に投げかける」仕組みだったのに対し、「導」では候補者の回答内容をリアルタイムで分析し、弱点を深掘りする質問を動的に生成します。これにより、人間の熟練した面接官と同等の面接体験を実現しました。

### 2. 33分野の専門面接

ソフトウェアエンジニアリング、データサイエンス、プロダクトマネジメント、UXデザイン、マーケティングなど、33の専門分野に対応。各分野の実際の面接で出題される内容を反映した技術テスト（選択問題5問＋記述問題5問）とビデオ面接を提供します。

### 3. 即時フィードバック

面接終了後、AIが回答内容を「技術的正確性」「深度と応用力」「構造化思考」「コミュニケーション」の4軸で評価。強みと改善ポイントを具体的にフィードバックし、次の面接に直結する改善アクションを提示します。

### 4. AIスキルマッチング

AI面接で測定されたスキルプロファイルをもとに、企業の求人と自動マッチング。経歴書だけでは見えない「実力」を可視化し、候補者と企業の最適なマッチングを支援します。

### 5. 完全無料・回数無制限

AI面接練習、スキル評価、求人マッチングなど、すべての基本機能を完全無料で提供。回数制限もなく、納得がいくまで何度でも練習できます。

## 企業向けAI採用支援

法人向けには、AIによる候補者スクリーニング・スキル評価サービスを提供しています。

- **採用コスト削減**: AIによる一次スクリーニングで、書類選考・一次面接の工数を大幅削減
- **選考期間の短縮**: AIが24時間365日面接を実施し、選考リードタイムを短縮
- **ミスマッチの防止**: 技術力だけでなく、コミュニケーション力・カルチャーフィットを多角的に評価

## 技術概要

- フロントエンド: Next.js (App Router), TypeScript, Tailwind CSS
- バックエンド: Next.js API Routes, Supabase (PostgreSQL)
- AI/ML: OpenAI GPT-4（質問生成・評価）, Whisper API（音声認識）
- ホスティング: Vercel
