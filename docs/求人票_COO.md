# 求人票: COO（最高執行責任者）

## 会社概要

**株式会社Michibiki（設立準備中）**

AI面接技術を活用したエキスパート人材マッチングプラットフォーム「導（Michibiki）」を開発・運営するスタートアップです。33分野×9業界グループの専門AI面接と、独自のChain-of-Thought評価技術により、日本の採用市場を変革します。

米国では同様のモデルを採用したMercorが評価額$10B（約1.5兆円）を達成しており、日本市場でのFirst Moverとして急成長を目指しています。

### 営業・パートナーシップ（30%）
- BtoB営業戦略の立案・実行
- 初期顧客（企業）の開拓・関係構築
- 業界パートナー（人材会社、専門家団体等）との連携
- 投資家リレーション・資金調達支援

### チーム構築（20%）
- 採用計画の策定・実行
- 組織文化の構築
- 外部アドバイザーネットワークの構築
